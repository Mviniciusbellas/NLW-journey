## 🖥️ Projeto
Esse é um projeto Web Responsivo de uma agenda virtual de rotinas para exibir atividades diarías.

## 🚀 Tecnologias 
Esse projeto foi desenvolvido durante o NLW (Journey) da Rocketseat com as seguintes tecnologias:

- HTML
- CSS
- JS
- Git e Github
